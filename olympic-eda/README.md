# 🏅 Olympic Athletes EDA

Exploratory Data Analysis on the Olympic Athletes dataset from Kaggle.

**Dataset:** 70,000+ rows of Olympic athlete participation data

**Libraries Used:** Pandas, NumPy, Matplotlib, Seaborn

## What this project covers
- Data loading and inspection
- Handling missing values and duplicates
- Outlier detection and treatment using IQR method
- Univariate and Bivariate analysis
- Visualizations on age, height, weight, gender and medal distribution

## Dataset Source
[Olympic Dataset - Kaggle](https://www.kaggle.com/datasets/bhanupratapbiswas/olympic-data)
