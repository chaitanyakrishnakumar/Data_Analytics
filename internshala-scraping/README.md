# 💼 Internshala Internship Data Analysis

Web scraping and EDA project analyzing internship listings from Internshala.

**Source:** [Internshala](https://internshala.com/internships)

**Libraries Used:** Requests, BeautifulSoup, Pandas, NumPy, Matplotlib, Seaborn

## What this project covers
- Web scraping 130 pages of internship data using BeautifulSoup
- Extracting fields like title, company, location, stipend, duration and skills
- Data cleaning — handling missing values and inconsistent formats
- EDA on stipend patterns across locations, roles and skills
- Identifying IT vs non-IT internship trends

## Key Findings
- Location and role type significantly affect stipend
- Actively hiring listings tend to offer competitive pay
- Python and data related roles dominate IT internship listings
