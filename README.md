# 📊 Data Analytics Projects

A collection of data analytics projects built through hands-on practice at Innomatics Research Labs.

Exploring **data cleaning**, **EDA**, **visualization**, and **web scraping**!

## Projects

### 🏅 [Olympic Athletes EDA](./olympic-eda)
Exploratory Data Analysis on 70,000+ rows of Olympic athlete data from Kaggle.
Covers missing value treatment, outlier detection, and statistical visualizations.

### 💼 [Internshala Internship Analysis](./internshala-scraping)
Web scraping 130 pages of Internshala internship listings using BeautifulSoup,
followed by EDA to identify stipend patterns across roles, locations and skills.

## Libraries Used
Pandas • NumPy • Matplotlib • Seaborn • BeautifulSoup • Requests

## Tools
